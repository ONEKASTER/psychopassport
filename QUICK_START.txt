📝 ПОДГОТОВКА К ПУБЛИКАЦИИ:

1. ЗАГРУЗИТЬ НА GITHUB
   → Перейти https://github.com/new
   → Создать репозиторий "psychopassport"
   → Загрузить psychopassport_v6_tg.html + README.md
   → Включить GitHub Pages (Settings → Pages → Deploy from main branch)
   → Ссылка будет: https://YOUR_USERNAME.github.io/psychopassport/psychopassport_v6_tg.html

2. СОЗДАТЬ TELEGRAM БОТА
   → Написать @BotFather в Telegram
   → /newbot
   → Получить TOKEN

3. ИНТЕГРИРОВАТЬ В БОТА (Python пример)
   ────────────────────────────
   from telegram import Update, WebAppInfo, KeyboardButton, ReplyKeyboardMarkup
   from telegram.ext import Application, CommandHandler, ContextTypes
   
   async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
       web_app = WebAppInfo(url="https://YOUR_USERNAME.github.io/psychopassport/psychopassport_v6_tg.html")
       button = KeyboardButton(text="🧠 ПСИХОПАСПОРТ", web_app=web_app)
       reply_markup = ReplyKeyboardMarkup([[button]])
       await update.message.reply_text("Пройдите тест:", reply_markup=reply_markup)
   
   app = Application.builder().token("YOUR_BOT_TOKEN").build()
   app.add_handler(CommandHandler("start", start))
   app.run_polling()
   ────────────────────────────
